# Most used commands

## Docker-compose

### Build images

```powershell
    > DevOps/build.ps1
```

```sh
    sh DevOps/build.sh
```

### Build a netcore image

```powershell
    > DevOps/build.ps1 <service-name ...>
```

```sh
    sh DevOps/build.sh <service-name ...>
```
